# Iron Clash Legends

Мультяшный средневековый прототип браузерной PvP-стратегии для Яндекс Игр.

## What is inside

- Бой на 3 линиях в реальном времени.
- Энергия до 10 очков.
- Карты бойцов: Рыцарь, Лучник, Мечник, Разбойник, Чародей.
- Заклинания: Огненный шар, Ледяная печать, Свет жизни, Гром, Боевой клич.
- Тренировка с ботом.
- Замки, здоровье баз, таймер, HP-бары и боевые эффекты.
- Безопасное подключение SDK Яндекс Игр через `yandex.js`.

## How to run

Открой `index.html` в браузере.

Для Яндекс Игр загружается содержимое этой папки как статическая HTML-игра. SDK уже подключен, игра отправляет сигнал готовности после старта.
